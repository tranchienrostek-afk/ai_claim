GraphRAG là phương pháp RAG nâng cao sử dụng knowledge graph để cải thiện độ chính xác truy vấn trên dữ liệu lớn, thường từ Microsoft và cộng đồng open-source.

## Top 10 Repos Hay Nhất

Dựa trên độ phổ biến, stars và tính năng cập nhật đến 2026, đây là 10 repository GitHub hàng đầu về GraphRAG.**aibase**+1

- **microsoft/graphrag** (github.com/microsoft/graphrag): Repo chính thức từ Microsoft, modular system xây graph từ text và query hiệu quả trên dataset lớn.[[github](https://github.com/microsoft/graphrag)]
- **circlemind-ai/fast-graphrag** (github.com/circlemind-ai/fast-graphrag): Phiên bản nhanh, tiết kiệm chi phí 6x so với gốc, hỗ trợ agent-driven retrieval.[[github](https://github.com/circlemind-ai/fast-graphrag)]
- **gusye1234/nano-graphrag** (github.com/gusye1234/nano-graphrag): Đơn giản, dễ hack, dùng cho medical data và hierarchical RAG.[[github](https://github.com/gusye1234/nano-graphrag)]
- **getzep/graphiti** (github.com/getzep/graphiti): Xây dựng dynamic knowledge graphs temporally-aware, fork phổ biến cho real-time query.[[github](https://github.com/graphrag)]
- **DEEP-PolyU/Awesome-GraphRAG** (github.com/DEEP-PolyU/Awesome-GraphRAG): Curated list benchmarks, papers và projects, tài liệu tham khảo tốt nhất.[[github](https://github.com/DEEP-PolyU/Awesome-GraphRAG)]
- **YZ-Cai/SimGRAG** (github.com/YZ-Cai/SimGRAG): Tập trung similar subgraphs cho KG-driven RAG, paper-based implementation.[[github](https://github.com/graphrag)]
- **LightRAG** (dựa nano-graphrag): Simple fast RAG với graph index, dùng Ollama local.**github**+1

- **GraphRAG-SDK** : Toolkit chuyên build GraphRAG systems, dễ tích hợp pipelines.[[github](https://github.com/DEEP-PolyU/Awesome-GraphRAG)]
